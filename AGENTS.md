# AGENTS.md

本文件适用于整个仓库，供参与本项目的人和自动化编码 Agent 共同遵循。

## 项目简介

这是一个无依赖的 Chrome Manifest V3 扩展，用于将当前页面保存到 Get笔记，并可选择归档到知识库。

## 开放协作

- 欢迎任何人参与修改、修复问题、完善文档或增加功能，无需事先申请。
- 提交改动时请说明目的、主要变化和验证方式，尽量保持改动小而清晰。
- 修改功能、权限、配置或使用方式时，请同步更新 `README.md`。
- 不要提交 API Key、Client ID 或其他个人凭据。
- 本项目采用 MIT License；参与贡献即表示同意贡献内容按该协议发布。

## 项目结构

- `manifest.json`：Chrome 扩展清单与权限声明。
- `src/`：弹窗、设置页、后台脚本和共享逻辑。
- `assets/`：扩展图标等静态资源。
- `tests/`：使用 Node.js 内置测试运行器的测试。

## 开发约定

- 保持项目无第三方运行时依赖，除非新增依赖确有必要并已在说明中写明原因。
- 遵循现有 JavaScript、HTML 和 CSS 风格，优先做小范围、可审阅的修改。
- Chrome 权限遵循最小权限原则；新增或扩大权限时必须同步说明用途与隐私影响。
- 不增加分析、遥测或额外网络请求，除非相关改动经过明确讨论。
- 为行为变化补充或更新测试；仅修改文档时可不运行功能测试。

## 验证

需要 Node.js 20 或更高版本。提交代码前运行：

```bash
npm test
npm run check
```

涉及界面或扩展行为的修改，还应在 Chrome 的扩展程序页面重新加载本项目并进行手动验证。
